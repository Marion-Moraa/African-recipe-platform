# African Recipe Sharing Platform

## Description
A web-based application that allows users to share and view African recipes using PHP, MySQL, HTML, and CSS.

## Features
- User registration and login
- Add and view recipes
- Story-based recipe UI
- Modern food app design

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## How to Run
1. Install XAMPP
2. Place project in htdocs
3. Start Apache & MySQL
4. Open: http://localhost/African_recipe_platform/
